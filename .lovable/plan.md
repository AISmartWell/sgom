
# Улучшение визуализации Automation Center

## Что будет сделано

Полная визуальная переработка страницы Automation Center с добавлением графиков, анимаций и интерактивных элементов для всех трёх вкладок.

## Изменения по вкладкам

### 1. Alerts & Monitoring
- **Статистические карточки** вверху: общее количество алертов, критические, предупреждения, информационные -- с иконками и цветовой кодировкой
- **Круговая диаграмма** (Recharts PieChart) распределения алертов по типу (Production Drop / Water Cut / Status Change)
- **Гистограмма** (Recharts BarChart) алертов по дням за последнюю неделю
- Сохранение текущего списка алертов с улучшенным оформлением

### 2. Auto-Linking
- **Анимированная блок-схема** процесса связывания: Core Image Upload --> Trigger --> Wells Table Match --> Auto-Link -- с CSS-анимированными стрелками/линиями между блоками
- **Статистика** связывания: количество связанных / несвязанных изображений (демо-данные)
- **Индикатор статуса триггера** с пульсирующей анимацией "Active"

### 3. Full Pipeline
- **Вертикальный таймлайн** вместо плоской сетки -- каждый этап как узел таймлайна с иконкой, названием и описанием
- При запуске пайплайна: этапы последовательно подсвечиваются с анимацией, пройденные получают зелёную галочку, текущий пульсирует
- **Прогресс-бар** с градиентом и процентом
- По завершении -- summary-карточка с результатами (демо)

## Технические детали

- Все графики через `recharts` (уже установлен)
- Анимации через Tailwind CSS (`animate-pulse`, `transition-all`, кастомные keyframes при необходимости)
- Один файл: `src/pages/modules/Automation.tsx` -- полная переработка компонента
- Демо-данные для графиков генерируются на клиенте (не требуют изменений БД)
- Существующая логика (загрузка алертов, realtime, pipeline simulation) сохраняется полностью
